== ARINC 429 Labels
The following sections describe the ARINC 429 labels communicated between the PC application
and device applications.
Unless otherwise specified the SDI field of the label is not used.
Unless otherwise specified the SSM field of the label has the following interpretation:

[cols="3*^", options="header,autowidth"]
|===
|SSM1 [Bit 30] |SSM0 [Bit 29] |Meaning
|0             |0             |Normal Operation (NO)
|0             |1             |No Computed Data (NCD)
|1             |0             |Functional Test (FT)
|1             |1             |Failure Warning (FW)
|===
