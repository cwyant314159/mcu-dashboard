== LCD Backlight [300~8~]

*Transmitter:* PC +
*Receiver:* Device +
*Description:* +
This label is sent from the PC to command the Device to enable LCD screen's backlight.
The Device sends this label back as status to the PC with a 500ms period.

2+|SDI

|===

*Backlight Enable*

* *0* - LCD Backlight off
* *1* - LCD Backlight on
